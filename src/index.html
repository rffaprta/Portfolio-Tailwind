Project Terbaru

Beberapa proyek yang telah saya kerjakan dengan fokus pada fungsionalitas dan efisiensi kode.

Movie App

Ongoing Project: React Movie App. Eksplorasi Front-End menggunakan React JS untuk membangun antarmuka modular yang interaktif. Proyek ini mengimplementasikan state management dan memecah UI menjadi komponen fungsional terpisah (seperti search panel, filter, dan add form) untuk menangani manipulasi data secara real-time.

Guessing Number Game

Game tebak angka berbasis JavaScript yang memanfaatkan logika kondisional untuk membandingkan tebakan pengguna dengan angka acak. Permainan ini dilengkapi sistem feedback real-time yang memberikan petunjuk arah tebakan, serta manipulasi DOM untuk memicu pesan kemenangan dan perubahan visual saat tebakan benar.

Simon Game

Game memori interaktif (Simon Game) yang dibangun sepenuhnya dengan Vanilla JavaScript. Proyek ini menonjolkan penggunaan Array untuk menghasilkan pola acak, manipulasi DOM untuk animasi tombol dan respons audio real-time, serta logika state management murni untuk mengatur alur permainan dari awal, naik level, hingga game over.

Expense Calculator

Aplikasi pencatat keuangan berbasis web untuk melacak pengeluaran harian secara real-time. Proyek ini mengandalkan manipulasi DOM JavaScript untuk validasi input pengguna, serta integrasi LocalStorage agar data riwayat keuangan tetap tersimpan aman meskipun browser ditutup atau di-refresh.

Project Warna

Tool UI interaktif untuk memanipulasi warna latar belakang website secara real-time menggunakan slider RGB. Proyek ini menonjolkan penggunaan event handling JavaScript untuk menangkap input pengguna dan memperbarui properti CSS secara dinamis tanpa perlu memuat ulang halaman.
